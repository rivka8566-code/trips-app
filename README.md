# 🏔️ Trips App - אפליקציית טיולים

אפליקציה לניהול וצפייה בטיולים, בנויה עם Angular.

## 📋 דרישות מקדימות

- Node.js (גרסה 18 ומעלה)
- npm (מותקן עם Node.js)

## 🚀 הרצת הפרויקט

### 1. התקנת תלויות

```bash
cd trips-app
npm install
```

### 2. הרצת השרת (JSON Server)

פתח טרמינל ראשון והרץ:

```bash
npx json-server --watch db.json
```

השרת יעלה על: `http://localhost:3000`

### 3. הרצת האפליקציה (Angular)

פתח טרמינל שני והרץ:

```bash
cd trips-app
npm start
```

האפליקציה תעלה על: `http://localhost:4200`
